## 💡 Customization Tips

### Colors
All colors are defined in CSS variables at the top of `style.css`. Simply modify these values to change the entire color scheme:

```css
:root {
  --primary-color: #00f5ff;
  --secondary-color: #7b2ff7;
  --accent-color: #ff006e;
}
```

### Typography
Font families can be changed by modifying the Google Fonts import and the font-family declarations:

```css
@import url('https://fonts.googleapis.com/css2?family=Your+Font');
body { font-family: "Your Font", sans-serif; }
```

### Animations
Animation speeds and easing functions can be adjusted in the transition properties:

```css
transition: all 0.4s cubic-bezier(0.4, 0, 0.2, 1);
```

### Effects
To reduce or enhance effects, adjust the opacity values and transform scales throughout the CSS.
